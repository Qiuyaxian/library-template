# library-template

library-template 是基于 webpack@4 与 vue2 组件库基础模版，主要适用于收录公司内部开发过程中常用的、带强业务内容而无常规替代方案的组件，也可按照公司内部需要选择适用基础 UI 库进行二次封装集成开发

## Support

- [x] 开箱即用
- [x] 快速生成组件
- [x] 支持组件案例演示
- [x] 支持 git 提交代码规范化，并生成 change-log
